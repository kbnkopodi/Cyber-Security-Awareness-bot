CyberBot - Cybersecurity Awareness Chatbot
CyberBot is a chatbot that runs in the console. It answers questions about cybersecurity topics, greets you by name, and reads its responses out loud.

What it does

Asks for your name and uses it throughout the chat
Answers questions about cybersecurity topics
Reads every response out loud using text to speech
Text types out character by character
Works no matter if you type in upper or lower case
Shows a cool ASCII art banner on startup


What you can ask it

how are you — general greeting
whats your purpose — what the bot does
what can i ask you about — lists all topics
phishing — explains what phishing scams are
safe browsing — how to stay safe on the internet
password safety — how to make and protect strong passwords
bye — exits the chat


What you need to run it

Windows PC
Visual Studio
.NET Framework 4.7.2 or higher


How to run it

Download or clone the project
Open it in Visual Studio
Press Ctrl + F5 to run


How to use it

Run the program
Type your name and press Enter
Type any topic from the list above
Type bye to exit


Built with

C#
.NET Framework
System.Speech for text to speech
